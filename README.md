# New-XrayR-Alpine

适用于Alpine系统的XrayR一键脚本

安装目录位于 /etc/XrayR

# 安装脚本

```shell script
apk add wget ; wget -O /usr/bin/xrayr https://raw.githubusercontent.com/mingge9527/New-XrayR-Alpine/main/xrayr.sh ; chmod 777 /usr/bin/xrayr ; xrayr
```
下次运行即可直接输入
```shell script
xrayr
```
